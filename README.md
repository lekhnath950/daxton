*Program:1* **Program to print a content using printf and puts**





```
// C program to show the use of puts

#include <stdio.h>

int main()
{

    puts("GNDEC");

    puts("LUDHIANA");
 

    getchar();

    return 0;
}
```

```
Output: 
GNDEC
LUDHIANA
```
